从城市纹理到指尖光华：巴西珠宝设计师的上海创业梦

澎湃新闻记者 王昱
2025-11-30 19:43
来源：澎湃新闻
∙ IP SH >
字号

在上海，城市的纹理会悄悄爬进你的心。

陆家嘴的四件套在天际线闪闪发亮，老建筑上的 Art Deco 花案沉静优雅，小区里盛开的山茶花低调而温柔——在巨鹿路的老洋房里，这些属于上海的动人符号，像一串细碎的光，串起了 Fernanda 的创作灵感，化作指间与颈畔的美丽。

从小在巴西长大的 Fernanda，与中国的联系始于母亲故乡的味道。外祖父母在巴西经营着一家中餐馆，家族对烹饪的热爱，让“食物中的中国”成为她最初、也是最温暖的文化记忆。

2012年，她第一次来到上海。Fernanda 很快在这座城市找到共鸣——上海既国际化又处处藏着传统的魅力。它的街区、建筑与日常细节不断吸引着她，最终推动她在这里开启自己的珠宝设计事业，也成为她创作灵魂真正的源泉。 

她的作品里，既有对春日漫步时悄然绽放的花苞的温柔凝望，也有对老建筑上被岁月打磨的纹样的回应，更有对三四十年代银幕上中国女性的致敬。骑着自行车缓缓穿行，是她与上海相处的方式——“只有骑得慢一点，你才有时间看见这座城市真正的美。”

对 Fernanda 来说，首饰不仅是装饰，更是“承载记忆的小小珍宝”。她常常被顾客的故事触动：有人因为她的设计开始对中国文化产生兴趣；有人把首饰当作纪念城市的礼物。这样的相遇，让她深切感受到——创作可以跨越距离，也能跨越语言。

在她眼里，上海是一座“便捷、活力、细腻”的城市。而在这座城市温柔的怀抱里，她找到了自己的方向、事业与归属感。

 巴西珠宝设计师 Fernanda （本文图片均为受访人提供）

IP SHANGHAI：是什么样的契机让您来到上海？

Fernanda：我母亲是华人，童年时期便随家人移民到巴西，父亲则是当地的巴西人。2012年，我在间隔年里来到上海，原本只是想学点中文，多了解一些中国文化，却没想到深深爱上了这里的生活。从那一刻起，一待便是十三年。

IP SHANGHAI：在来中国之前，您对上海这座东方都市有着怎样的印象？来到这里之后，又有哪些不同的感受？

Fernanda： 我妈妈从小就在巴西长大，对中国的了解其实非常有限。对我来说，与中国联系最紧密的部分反而是“食物”。我的外祖父母在巴西经营着一家中餐馆，而我们家每个人都特别喜欢做饭，所以饮食一直是我成长中很重要的一部分。

来到上海之后，我发现这里真的很特别。它既是一座高度国际化、充满活力的城市，同时又随处可见那些保留着浓厚中国特色的传统小角落——这些对我来说都特别有吸引力。 

IP SHANGHAI：中国和巴西文化之间有什么相通之处？

Fernanda：实际上，巴西是一个由众多移民构成的国家。在我生活的城市里，汇聚了来自中国、日本、黎巴嫩、葡萄牙、西班牙、意大利等各地的文化与族群。如果要说两国的相似之处，我觉得巴西人与中国人一样，都热情开朗、非常接地气。

IP SHANGHAI：为什么选择在上海开创珠宝设计的事业？

Fernanda：我在巴西读书学的就是珠宝和配饰设计，后来又去了米兰攻读设计硕士。在上海创业，可以说是以一种非常小而谦逊的方式开始的，因为刚开始的时候，我注意到上海有很多小型的设计市集和快闪活动，许多人都在经营自己的小品牌。那时我心想，也许我也可以成为其中一员。

这种创业精神很纯粹，没有宏大的商业计划，一切都源自一个小小的梦想——只是想试试看会发生什么。没想到，随着一年又一年的积累，我慢慢把业务理顺，建立了自己的客户群——那些真正喜欢我作品的人。渐渐地，一切也开始步入正轨。

在巴西学习珠宝和配饰设计的Fernanda决定在上海创立自己的品牌。 

IP SHANGHAI：为什么在珠宝设计中加入上海元素？有什么故事吗？

Fernanda： 就像我创立品牌一样，这其实更多是一种潜意识的选择——看到自己真正喜欢的东西，自然而然就想把它们做成作品。上海对我有一种特别的吸引力，它的历史、故事和文化都让我着迷，总是想去深入探索。

比如我的“女演员系列”，就是向上世纪三四十年代中国影视中的女演员和女编剧致敬。我还做了一些小研究，比如读张爱玲。另外还有“茶花系列”，这个系列背后也有很有趣的故事。我特别喜欢春天，喜欢大自然，那种生机勃勃的感觉让我很快乐。而茶花虽然低调，却随处可见，它们不张扬，却让人心生温暖，每次看到小区里的老奶奶们精心照料茶花的画面，就让我觉得很甜美。

更有意思的是，在一些上海老建筑上，也能看到类似茶花或玫瑰的装饰图案，它们的形状很像中国文化中的祥云。我觉得这些元素带来的不仅是美感，还有快乐与振奋，于是我就有了把它们设计进珠宝的想法。

山茶花、白玉兰、还有上海建筑上的纹理都成为了Fernanda创作的源泉。

IP SHANGHAI：很好奇您的工作习惯是怎样的？又是如何发现那些图案的？

Fernanda：我的灵感大多来自日常生活，其实非常随意——大部分时间我只是随便在城市里走走。每当看到漂亮的东西，我会拍照，或者在脑子里记下来。很多灵感都是这样自然而然地出现的，并没有固定计划，比如“今天一定要去某个地方”。之后，我会不断画草图，尝试去理解和转化，因为首饰不仅仅是漂亮的图纸，它还要真正成为人们可以佩戴的作品。

还有一点非常重要——我骑自行车。搬到上海时我买的第一件东西之一就是自行车，到现在已经十三年了。通常我就是靠它在城市里四处穿行。我觉得，当你骑得慢一些的时候，才能真正环顾四周，慢慢发现，这座城市其实非常美丽。 

Fernanda喜欢骑行，在日常生活中发现美的事物。

IP SHANGHAI：通过珠宝，您希望传递给欣赏作品的人怎样的讯息？

Fernanda：我觉得珠宝是一件非常私人的作品。虽然是我创造的，但当人们购买之后，他们会用它去创造属于自己的故事，比如，朋友们会买同款饰品，变成“情侣款”或“闺蜜款”，这样一来，首饰就承载了情感的意义。有的人是收到礼物而获得，有的人则是犒劳自己，或者纪念某个特别的时刻——我也有一些客人是为了庆祝人生中特定的重要时刻而来挑选珠宝。

我特别喜欢首饰的一点是——它当然是一种配饰，可以让人更好看、更有气质，戴上它也是为了让自己更得体。但对我来说，首饰还有非常感性的一面，它像一件“珍宝”，承载着意义与记忆。所以我的作品真正想传递的，是对日常生活的珍惜。

很多时候，人们一想到快乐、幸福或特别的时刻，总觉得它们遥远而宏大，好像永远在未来。而我希望通过自己的品牌，把这种感受拉回到当下，让人们在此时此地就能体验喜悦与珍惜，真正去在意、去呵护这些“珍宝”。在这个一切快速更替、被轻易丢弃的时代，拥有少数你真正珍视、并能长久陪伴的事物，是一件非常特别的事情——对我而言，这正是珠宝最迷人的地方。 

Fernanda希望让人们能体验喜悦与珍惜，真正去在意、呵护自己的“珍宝”。 

IP SHANGHAI：您的客人大概是怎样的人群？有没有印象深刻的故事？

Fernanda：我的大部分客人都是在上海生活的人，其中很多是外国人，也有本地客人或居住在海外的中国人。我经常收到他们的反馈，说佩戴我的首饰多年后，它们真的成了“记忆之物”。尤其是那些已经离开中国的人，这些首饰成了他们的纪念品，让他们回想起在这里的生活。

去年，有一件事让我特别开心。一次，我在安福路偶遇一位客人，她当时和婆婆一起。她告诉我，她买过我的一件“窗”系列首饰作为礼物。她说，这件带有中国元素的首饰，也成为她婆婆对中国产生兴趣的契机之一，让她产生了想来中国、想了解上海的好奇心。我的作品恰好点燃了这个小小的火花，我真的很感动。

通过Fernanda的作品，很多外国朋友对中国产生了兴趣。 

IP SHANGHAI：在上海创业难吗？刚开始做品牌的时候，有遇到什么困难吗？

Fernanda：我觉得整体来看，中国对创业者有很多支持和优惠政策，确实提供了尝试创业的空间。上海也有很多小型创业者，大家会互相分享经验、互相帮助，所以起步其实并没有那么困难，手续办理方面也比较顺利。总的来说，我觉得上海是一个非常鼓励人们开创自己事业的城市。

IP SHANGHAI：对于那些想在上海创业的独立设计师，无论是外国人还是本地人，您有什么建议吗？

Fernanda：我有两个建议。第一，一定要一步一步来，把事情拆成每天都能完成的小目标。长期坚持下来，会更容易起步，也更容易坚持下去。很多时候，人们会被“大局”吓到，觉得事情太大、太复杂，不知道从哪里开始。但如果从小处着手，会轻松很多。第二，如果有机会学习一些商业、管理或市场营销方面的知识，即便只是一点点，也会对你特别有帮助。

创业一定要一步一步来，把事情拆成每天都能完成的小目标。

IP SHANGHAI：如果要向外国朋友介绍上海，你会用哪些词来形容这座城市？

Fernanda：我觉得上海是一座非常有活力的城市。中国人经常用“方便”来形容城市，而这真的一点也不夸张。在巴西，我们几乎不会用“方便”这个词来评价一座城市，甚至很少把它当作城市的特质。但上海正是一座方便的城市，无论生活还是出行，都非常顺畅。

另外，我特别喜欢这里的建筑。在那些街区走过时，我会突然意识到：原来我生活在这样一座细腻而有故事的城市里。
